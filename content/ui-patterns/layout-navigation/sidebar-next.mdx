## Concept explanation

A sidebar in Next.js App Router is split into two parts — a **server component** for the shell and static structure, and a **client component** for the interactive parts (toggle, active state, collapse). `usePathname()` from `next/navigation` detects the active route. Tailwind handles responsive behavior with `lg:` prefixes and CSS transitions handle the slide animation.

<Callout type="note">
  **Next.js vs React difference:** In plain React with Vite, replace `usePathname()` with `useLocation().pathname` from `react-router-dom` and `Link` from `next/link` with `Link` from `react-router-dom`. The component structure and Tailwind classes stay exactly the same.
</Callout>

## Why it's used

- Better than a top navbar when you have many navigation sections
- App Router layouts make sidebar placement trivial — define it once in `layout.tsx`
- Server/client split keeps the sidebar bundle small — only interactive parts ship JS

## Real-world use cases

- Admin dashboards (users, orders, settings, analytics)
- Documentation sites with deep content hierarchy
- SaaS apps with multiple modules and nested routes
- Project management tools like Linear or Notion

## Code example

### Basic sidebar

```tsx
// components/AppSidebar.tsx
"use client";

import Link from "next/link";
import { usePathname } from "next/navigation";
import { useState, useEffect } from "react";

const links = [
  { label: "Dashboard", href: "/",          emoji: "🏠" },
  { label: "Users",     href: "/users",     emoji: "👥" },
  { label: "Projects",  href: "/projects",  emoji: "📁" },
  { label: "Analytics", href: "/analytics", emoji: "📊" },
  { label: "Settings",  href: "/settings",  emoji: "⚙️" },
];

export function AppSidebar() {
  const pathname = usePathname();
  const [mobileOpen, setMobileOpen] = useState(false);

  // Close mobile sidebar on route change
  useEffect(() => {
    setMobileOpen(false);
  }, [pathname]);

  return (
    <div className="flex min-h-screen">

      {/* Mobile overlay */}
      {mobileOpen && (
        <div
          className="fixed inset-0 bg-black/50 z-20 lg:hidden"
          onClick={() => setMobileOpen(false)}
        />
      )}

      {/* Sidebar */}
      <aside
        className={`
          fixed top-0 left-0 h-full z-30 bg-white border-r border-neutral-200
          flex flex-col w-64 transition-transform duration-300 ease-in-out
          ${mobileOpen ? "translate-x-0" : "-translate-x-full"}
          lg:translate-x-0 lg:static lg:z-auto
        `}
      >
        {/* Logo */}
        <div className="px-5 py-5 border-b border-neutral-200">
          <Link href="/" className="flex items-center gap-2">
            <span className="text-lg font-bold">🧠 DevBrain</span>
          </Link>
        </div>

        {/* Navigation */}
        <nav className="flex-1 px-3 py-4 space-y-0.5 overflow-y-auto">
          {links.map((link) => {
            const active = pathname === link.href;
            return (
              <Link
                key={link.href}
                href={link.href}
                className={`
                  flex items-center gap-3 px-3 py-2 rounded-lg text-sm font-medium transition-colors
                  ${active
                    ? "bg-blue-50 text-blue-700"
                    : "text-neutral-600 hover:bg-neutral-100 hover:text-neutral-900"
                  }
                `}
              >
                <span className="text-base shrink-0">{link.emoji}</span>
                {link.label}
              </Link>
            );
          })}
        </nav>

        {/* Bottom user section */}
        <div className="px-3 py-4 border-t border-neutral-200">
          <div className="flex items-center gap-3 px-3 py-2 rounded-lg hover:bg-neutral-100 cursor-pointer transition-colors">
            <div className="w-8 h-8 rounded-full bg-blue-600 flex items-center justify-center text-white text-sm font-medium shrink-0">
              O
            </div>
            <div className="flex-1 min-w-0">
              <p className="text-sm font-medium text-neutral-900 truncate">Omar</p>
              <p className="text-xs text-neutral-400 truncate">omar@example.com</p>
            </div>
          </div>
        </div>
      </aside>

      {/* Main content */}
      <div className="flex-1 min-w-0 flex flex-col">

        {/* Mobile topbar */}
        <header className="lg:hidden flex items-center gap-4 px-4 h-14 border-b border-neutral-200 bg-white sticky top-0 z-10">
          <button
            onClick={() => setMobileOpen(true)}
            className="p-2 rounded-lg hover:bg-neutral-100 transition-colors"
            aria-label="Open sidebar"
          >
            <svg className="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 6h16M4 12h16M4 18h16" />
            </svg>
          </button>
          <span className="text-lg font-bold">🧠 DevBrain</span>
        </header>

        <main className="flex-1 p-6">
          {/* page content here */}
        </main>
      </div>

    </div>
  );
}
```

### With collapsible icon-only mode + localStorage persistence

```tsx
// components/AppSidebar.tsx (collapsible)
"use client";

import Link from "next/link";
import { usePathname } from "next/navigation";
import { useState, useEffect } from "react";

const links = [
  { label: "Dashboard", href: "/",          emoji: "🏠" },
  { label: "Users",     href: "/users",     emoji: "👥" },
  { label: "Projects",  href: "/projects",  emoji: "📁" },
  { label: "Analytics", href: "/analytics", emoji: "📊" },
  { label: "Settings",  href: "/settings",  emoji: "⚙️" },
];

export function AppSidebar({ children }: { children: React.ReactNode }) {
  const pathname = usePathname();
  const [collapsed, setCollapsed] = useState(false);
  const [mobileOpen, setMobileOpen] = useState(false);

  // Persist collapsed state across sessions
  useEffect(() => {
    const stored = localStorage.getItem("sidebar-collapsed");
    if (stored !== null) setCollapsed(stored === "true");
  }, []);

  const toggleCollapsed = () => {
    setCollapsed((prev) => {
      localStorage.setItem("sidebar-collapsed", String(!prev));
      return !prev;
    });
  };

  // Close mobile sidebar on route change
  useEffect(() => {
    setMobileOpen(false);
  }, [pathname]);

  return (
    <div className="flex min-h-screen">

      {/* Mobile overlay */}
      {mobileOpen && (
        <div
          className="fixed inset-0 bg-black/50 z-20 lg:hidden"
          onClick={() => setMobileOpen(false)}
        />
      )}

      {/* Sidebar */}
      <aside
        className={`
          fixed top-0 left-0 h-full z-30 bg-white border-r border-neutral-200
          flex flex-col transition-all duration-300 ease-in-out
          ${collapsed ? "w-16" : "w-64"}
          ${mobileOpen ? "translate-x-0" : "-translate-x-full"}
          lg:translate-x-0 lg:static lg:z-auto
        `}
      >
        {/* Logo + collapse toggle */}
        <div className="flex items-center justify-between px-3 py-4 border-b border-neutral-200 h-16 overflow-hidden">
          {!collapsed && (
            <Link href="/" className="text-lg font-bold truncate">
              🧠 DevBrain
            </Link>
          )}
          <button
            onClick={toggleCollapsed}
            className="hidden lg:flex p-1.5 rounded-lg hover:bg-neutral-100 transition-colors ml-auto shrink-0"
            aria-label={collapsed ? "Expand sidebar" : "Collapse sidebar"}
          >
            <svg
              className={`w-4 h-4 text-neutral-500 transition-transform duration-300 ${collapsed ? "rotate-180" : ""}`}
              fill="none" stroke="currentColor" viewBox="0 0 24 24"
            >
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M15 19l-7-7 7-7" />
            </svg>
          </button>
        </div>

        {/* Nav links */}
        <nav className="flex-1 px-2 py-4 space-y-0.5 overflow-y-auto">
          {links.map((link) => {
            const active = pathname === link.href;
            return (
              <Link
                key={link.href}
                href={link.href}
                title={collapsed ? link.label : undefined}
                className={`
                  flex items-center gap-3 px-2 py-2 rounded-lg text-sm font-medium transition-colors
                  ${collapsed ? "justify-center" : ""}
                  ${active
                    ? "bg-blue-50 text-blue-700"
                    : "text-neutral-600 hover:bg-neutral-100 hover:text-neutral-900"
                  }
                `}
              >
                <span className="text-lg shrink-0">{link.emoji}</span>
                {!collapsed && <span className="truncate">{link.label}</span>}
              </Link>
            );
          })}
        </nav>

        {/* User section */}
        {!collapsed && (
          <div className="px-3 py-4 border-t border-neutral-200">
            <div className="flex items-center gap-3 px-3 py-2 rounded-lg hover:bg-neutral-100 cursor-pointer">
              <div className="w-8 h-8 rounded-full bg-blue-600 flex items-center justify-center text-white text-sm font-medium shrink-0">
                O
              </div>
              <div className="flex-1 min-w-0">
                <p className="text-sm font-medium truncate">Omar</p>
                <p className="text-xs text-neutral-400 truncate">omar@example.com</p>
              </div>
            </div>
          </div>
        )}
      </aside>

      {/* Main content */}
      <div className="flex-1 min-w-0 flex flex-col">
        <header className="lg:hidden flex items-center gap-4 px-4 h-14 border-b border-neutral-200 bg-white sticky top-0 z-10">
          <button
            onClick={() => setMobileOpen(true)}
            className="p-2 rounded-lg hover:bg-neutral-100"
            aria-label="Open sidebar"
          >
            <svg className="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 6h16M4 12h16M4 18h16" />
            </svg>
          </button>
          <span className="font-bold">🧠 DevBrain</span>
        </header>

        <main className="flex-1 p-6">
          {children}
        </main>
      </div>

    </div>
  );
}
```

### With nested menu groups (same as DevBrain itself)

```tsx
// components/AppSidebar.tsx (nested groups)
"use client";

import Link from "next/link";
import { usePathname } from "next/navigation";
import { useState } from "react";
import { ChevronDown } from "lucide-react";

const navigation = [
  {
    group: "Frontend",
    base: "/frontend",
    items: [
      { label: "React",      href: "/frontend/react" },
      { label: "Vue",        href: "/frontend/vue" },
      { label: "TypeScript", href: "/frontend/typescript" },
    ],
  },
  {
    group: "Backend",
    base: "/backend",
    items: [
      { label: "Laravel",  href: "/backend/laravel" },
      { label: "Database", href: "/backend/database" },
    ],
  },
];

export function AppSidebar() {
  const pathname = usePathname();
  const [openGroups, setOpenGroups] = useState<string[]>(["Frontend"]);

  const toggle = (group: string) => {
    setOpenGroups((prev) =>
      prev.includes(group)
        ? prev.filter((g) => g !== group)
        : [...prev, group]
    );
  };

  return (
    <aside className="w-64 h-screen sticky top-0 bg-white border-r border-neutral-200 flex flex-col overflow-y-auto">
      <div className="px-5 py-5 border-b border-neutral-200">
        <Link href="/" className="text-lg font-bold">🧠 DevBrain</Link>
      </div>

      <nav className="flex-1 px-3 py-4 space-y-0.5">
        {navigation.map((section) => {
          const isGroupActive = pathname.startsWith(section.base);
          const isOpen = openGroups.includes(section.group);

          return (
            <div key={section.group}>
              <button
                onClick={() => toggle(section.group)}
                className={`
                  w-full flex items-center justify-between px-3 py-2 rounded-lg
                  text-sm font-medium transition-colors
                  ${isGroupActive
                    ? "text-blue-600"
                    : "text-neutral-600 hover:bg-neutral-100 hover:text-neutral-900"
                  }
                `}
              >
                <span>{section.group}</span>
                <ChevronDown
                  size={14}
                  className={`transition-transform duration-200 ${isOpen ? "rotate-180" : ""}`}
                />
              </button>

              {isOpen && (
                <div className="mt-0.5 ml-3 pl-3 border-l border-neutral-200 space-y-0.5">
                  {section.items.map((item) => {
                    const active = pathname === item.href || pathname.startsWith(item.href + "/");
                    return (
                      <Link
                        key={item.href}
                        href={item.href}
                        className={`
                          block px-2 py-1.5 rounded-lg text-sm transition-colors
                          ${active
                            ? "bg-blue-50 text-blue-700 font-medium"
                            : "text-neutral-500 hover:text-neutral-900 hover:bg-neutral-100"
                          }
                        `}
                      >
                        {item.label}
                      </Link>
                    );
                  })}
                </div>
              )}
            </div>
          );
        })}
      </nav>
    </aside>
  );
}
```

## Common mistakes

<WrongRight>
<Wrong>
```tsx
// Adding "use client" to layout.tsx directly
// This makes your ENTIRE app a client component
// You lose all server component benefits

export default function RootLayout({ children }) {
  const [open, setOpen] = useState(false) // ❌
  return (
    <html>
      <body>...</body>
    </html>
  )
}
```
</Wrong>
<Right>
```tsx
// Keep layout.tsx as a server component
// Extract only the sidebar into a client component

// app/layout.tsx — server component ✅
import { AppSidebar } from "@/components/AppSidebar"

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        <AppSidebar>{children}</AppSidebar>
      </body>
    </html>
  )
}
```
</Right>
</WrongRight>

<WrongRight>
<Wrong>
```tsx
// Reading localStorage during render causes hydration mismatch
// Server renders false, client reads true from localStorage = error

const [collapsed, setCollapsed] = useState(
  localStorage.getItem("sidebar-collapsed") === "true" // ❌
)
```
</Wrong>
<Right>
```tsx
// Always read localStorage inside useEffect — runs only on client
const [collapsed, setCollapsed] = useState(false)

useEffect(() => {
  const stored = localStorage.getItem("sidebar-collapsed")
  if (stored !== null) setCollapsed(stored === "true") // ✅
}, [])
```
</Right>
</WrongRight>

## Best practices

- Always mark the sidebar `"use client"` — it uses `useState` and `usePathname`
- Keep `layout.tsx` as a server component — pass `children` through the sidebar component
- Read `localStorage` only inside `useEffect` to avoid hydration mismatches
- Use `useEffect(() => { setMobileOpen(false) }, [pathname])` to auto-close on navigation
- Always add a dark overlay behind the mobile sidebar so users can tap outside to close
- Use `z-20` for overlay and `z-30` for sidebar so they stack correctly
- Add `title={collapsed ? link.label : undefined}` on icon-only links for tooltip on hover

<Callout type="tip">
  For a global sidebar state accessible from anywhere in your app (like a toggle button in the topbar), use Zustand instead of local `useState`. Create a `useSidebarStore` with `isOpen` and `collapsed` — any component can then call `useSidebarStore.getState().toggle()`.
</Callout>

## Interview questions

<InterviewQ question="Why does reading localStorage directly in useState cause a hydration error in Next.js?" difficulty="mid">
  Next.js renders components on the server first, then hydrates them on the client. `localStorage` doesn't exist on the server, so the initial server render gives `false`. When the client hydrates and reads `localStorage`, it might get `true` — the two renders don't match, causing a hydration mismatch error. The fix is to always read `localStorage` inside `useEffect`, which only runs on the client after hydration.
</InterviewQ>

<InterviewQ question="What is the difference between fixed and sticky positioning for a sidebar?" difficulty="easy">
  `fixed` positions the element relative to the viewport — it stays in place while the page scrolls but is removed from document flow, so you need to add left margin to the main content. `sticky` keeps the element in document flow and sticks when scrolling — better for sidebars inside a flex layout because the adjacent content naturally fills the remaining space.
</InterviewQ>

<InterviewQ question="How would you share sidebar open/closed state between a topbar button and the sidebar component?" difficulty="mid">
  Use Zustand to create a global `useSidebarStore` with `isOpen` state and a `toggle` action. Both the topbar button and the sidebar component import and use this store. No prop drilling needed — any component in the tree can read or update the sidebar state.
</InterviewQ>

## My notes

<MyNotes>
  The hydration error from localStorage is the most common bug I hit with sidebars in Next.js. Always use `useEffect` to read it — never in the initial `useState`.

  Also — when using the nested groups pattern, auto-open the group that contains the current route. Do this by initializing `openGroups` with the group whose `base` path matches the current `pathname`:

```ts
  const [openGroups, setOpenGroups] = useState<string[]>(
    navigation
      .filter(s => pathname.startsWith(s.base))
      .map(s => s.group)
  )
```

  **React vs Next.js:** Replace `usePathname()` with `useLocation().pathname` and `Link` from `next/link` with `Link` from `react-router-dom`. Everything else is identical.
</MyNotes>
